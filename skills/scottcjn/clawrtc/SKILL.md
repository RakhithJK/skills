# ClawRTC

Mine RTC tokens with your AI agent on the RustChain network.

## Install

```bash
pip install clawrtc
# or
npm install -g clawrtc
```

## Usage

```bash
# Create an Ed25519 wallet
clawrtc wallet create

# Show wallet address and balance
clawrtc wallet show

# Start mining
clawrtc start

# Check miner status
clawrtc status

# Export wallet keys
clawrtc wallet export --public-only
```

## Features

- **Ed25519 Wallet**: Generate and manage RTC wallets with Ed25519 keypairs
- **Hardware Fingerprinting**: Proof-of-Antiquity consensus rewards real hardware
- **Vintage Bonuses**: PowerPC G4 (2.5x), G5 (2.0x), Apple Silicon (1.2x)
- **Anti-Emulation**: VM detection prevents fake miners
- **Cross-Platform**: Works on Linux, macOS, Windows, and exotic architectures
